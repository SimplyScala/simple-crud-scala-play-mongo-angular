Simple CRUD

A Scala / Play / Mongo / AngularJS app that demonstrate how to write a basic app using modern technologies

On backend

This app is using Scala 2.10.0 and Play framework 2.2.1 to run the backend with MongoDB and ReactiveMongo 0.10.2 to store data.

There is several ways to implement a simple REST API and some are experimented here.

On frontend

This app is using standard frontend web technologies such as Bootstrap and AngularJS. Frontend tools are also used and integrated to be as productive as possible sur as Grunt and Bower.

Resources used to build this project

  • eventual activator from Typesafe
  • Reactive app from Stephane Godbillon
  • JSON Coast-to-Coast Design from Mandubian
